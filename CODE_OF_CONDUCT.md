# Code of Conduct

ChatDome expects project discussions to stay technical, specific, and respectful.

## Expected Behavior

- Discuss code, behavior, risks, and tradeoffs directly.
- Keep issue reports reproducible and actionable.
- Respect maintainer decisions on scope, security posture, and release timing.
- Avoid sharing secrets, private logs, tokens, or host-specific sensitive data.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language
- Publishing another user's secrets or private operational data
- Repeated off-topic comments after maintainers have redirected the discussion
- Publicly posting exploit details for unresolved security issues

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block accounts; or restrict participation when behavior harms the project or its users.
